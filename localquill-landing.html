Setup

Running in
4 steps.

One-time setup. Works on Windows, Mac, and Linux.

01
Install Ollama

Download and install Ollama from ollama.com. It's the local AI runtime that powers everything. Free and open source.

ollama.com/download
5 min
02
Pull a model

Download a model like Gemma 3 or Llama 3. Run this once in your terminal. The model downloads to your machine.

ollama pull gemma3
~3 GB
03
Start Ollama with CORS enabled

Run Ollama with this one-time environment variable so the extension can connect. Set it permanently in System Environment Variables.

OLLAMA_ORIGINS=* ollama serve
Once
04
Install LocalQuill

After purchase you'll get a Chrome Web Store link. Click Add to Chrome. Done. Open any page, highlight text, and start writing.

chrome.google.com/webstore/...

FAQ

Questions.

Do I need to pay for an AI API? +
No. LocalQuill uses Ollama which runs AI models locally on your machine. There are zero API costs, zero usage limits. You download the model once and it runs forever for free.
What models does it work with? +
Any model you have installed in Ollama — Gemma 3, Gemma 4, Llama 3, Mistral, Phi, Qwen, and more. LocalQuill auto-detects all your installed models and lets you pick.
Does it work on Windows? +
Yes. LocalQuill is a Chrome extension so it works on any OS — Windows, Mac, or Linux — as long as you have Chrome, Edge, or Brave installed.
Is my data really private? +
100%. Your text never leaves your machine. The extension communicates only with Ollama running at localhost. No analytics, no logging, no cloud processing.
